	- all the game files character/levels/etc
	  are located on data folder directly or 
	  inside AOF.pak if you are using the pak version.

	- If you wish to see or edit the pak
	  content do the two steps below

	1 - to unpak drag the file AOF.pak
	    on top of paxplode.bat

	2 - next cut/paste the data 
	    folder to the game folder
	    were AOF.exe is located

	3 - (optional) 
	    open the folder Paks/unpck copy AOF.pak
	    to data/paks and replace the original AOF.pak
	    if you dont to this the game will take 2x loading
	    like it was loading 2 games at the same time
